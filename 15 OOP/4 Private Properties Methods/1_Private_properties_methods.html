Private Properties and Methods

  1. Using keywords(var or let or const), you can create private properties and methods.
    Example:
    this.price //It's public.
    var price //It's private.
    let price //It's private.
  2. Example:
    i) In the following example var price = 300 is private. Therefore, it cannot be accessed from outside the scope.
    ii) var data = function() is a private method.
    var Mobile = function(model_no, sprice)
    {
    this.model = model_no;
    this.color = 'white';
    var price = 3000;
    this.sp = sprice;
    this.sellingprice = function()
    {
    return(this.price + this.sp);
    };
    var data = function()
    {
    document.write('Model No: '+this.model+' Price: '+this.sellingprice());
    };
    }
