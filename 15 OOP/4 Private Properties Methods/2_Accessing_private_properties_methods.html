Accessing Private Properties and Methods from outside the scope

  1. Using keywords(var or let or const), you can create private properties and methods.
    Example:
    this.price //It's public.
    var price //It's private.
    let price //It's private.
  2. Example:
    1. In the following example var price = 300 is private. Therefore, it cannot be accessed from outside the scope.
    2. var data = function() is a private method. var Mobile = function(model_no, sprice)
    3. If we try to access a private property from outside, the result will be undefined.
    4. If we try to access a method from outside, the result in the console will be 'method_name is not a function'

    JavaScripts Codes:
     var Mobile = function(model_no, sprice)
    {
    this.model = model_no;
    this.color = 'white';
    var price = 3000;
    this.sp = sprice;
    this.sellingprice = function()
    {
    return(this.price + this.sp);
    };
    var data = function()
    {
    document.write('Model No: '+this.model+' Price: '+this.sellingprice());
    };
    }
    var samsung = new Mobile('Galaxy', 2000);
    var nokia = new Mobile('3310', 1000);
    nokia.data();
