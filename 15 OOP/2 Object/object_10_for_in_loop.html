Object: for.....in loop

  • 1. for.....in loop is used to loop through an object's property. It means, if you want to visit the properties or want to see the keys, the for...in loop is used.
    syntax:
    for(var/let variable_name in object_name)
    {
    block statements;
    }
    eg:
    for(var specs in samsung)
    {
    document.write(specs);
    }
