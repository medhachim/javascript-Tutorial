Creating Object: Summary

  • 1. Using object literal:creates only one object.
    var fees = {};//Declaration
    fees['Rahul']=200;//Initialization
  • 2. Object literal declare and initialize:creates only one object.
    var fees = {Rahul:100};
  • 3. Object Constructor:creates only one object.
    var fees = newObject();
  • 4. Factory Function: Creating object from an object
    function mobile()//mobile is an object
    {
    return

    model:'Galaxy';
    }
    }
    var samsung = mobile()//samsung is an object
    var lg = mobile()//lg is an object
