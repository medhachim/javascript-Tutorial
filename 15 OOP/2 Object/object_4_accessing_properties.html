Accessing object properties

  • A property of an object is some piece of named data it contains. These are accessed with dot operator applied to an object alternative to the dot operator is the array [] operator.
    It means we can access the property using dot or array[] operator.
    Syntax:
    object_name.property_name
    eg:
    var fees = {Rahul: 100, Summit: 200, Rose: 2000,"Rock Lama": 5000};//declaration and initialization
    var fees = {};//Declaration
    fees['Rahul'] = 100; or fees.Rahul = 100; //Initialization
    Accessing the value of the property
    document.write(fees['Rahul']); // object and key
    document.write(fees.Rahul); //with dot operator
    document.write(fees['Rock Lama']); // cannot use dot operator for two word key.
