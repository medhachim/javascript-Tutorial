Object Method

  • Object members that are functions are called methods. These are accessed with dot operator applied to an object alternative to the dot operator is the array[] operator.
    syntax:
    object_name.Method_name();
    eg:
    var fees = {Rahul:100, Summit: 500, total:function(){return(100+200+300);}};
    The object fees has three members. First two are properties and the last is the method.
    or
    var fees = {};//Declaration
    //now, initialization
    fees.Rahul = 100;
    fees.Summit = 500;
    fees.total = function(){return(100+200+300);};
    //Accessing the method
    document.write(fees.total());//or document.write(fees["total"]);
  • Adding Properties/Method
    Syntax:
    Object_name.Property_name = value;
    or
    Object_name['Property_name'] = value;
    fees.Sonam = 500;
    or
    fees['Sonam'] = 500;
