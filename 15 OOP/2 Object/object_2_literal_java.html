OOP: Object Literal

  • Declaration and Initialization of objects
    There two methods to declare and initialize and object.
    1. Using object literal
    2. Using object constructor
    Syntax: var object_name = {};
    We should follow the naming rules for an object. Reserved keyword are not used.
    Eg:
    var fees = {};// var is a keyword, fees name of the object
    //It's an empty object as it has no properties or values.
    To assign the property to an object:
    object_name['key'] = value;

    fees = ['Rose'] = 100;// a property


    >the other way using dot operator: fees.Rose = 100;

    fees = ['Summit'] = 200;// a property


    >the other way using dot operator: fees.Summit = 200;

    fees = ['Rozie'] = 400;// a property

    >the other way using dot operator: fees.Rozie = 400;



    >Multiword key requires double quotes.
    eg:

    fees = ["Rose Tamang"] = 700;// cannot be written with dot operator because of the space that appears between the key.


    >Using object constructor

    var fees = new Object();

  • You can assign a function to the value of an object.

    fees['total'] = function() {return(100+200+400);};//It's a method (anonymous function).
    docurment.write(fees['total']());


    function sum()
    {
    return(100+200+400);
    }

    fees['total'] = sum;//Here the function used as a value. or fees.total = sum;
