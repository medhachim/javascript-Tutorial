OOP: Object Constructor

    Declaration and initialization of object using object constructor.
     syntax: var object_name = new Object();
    //new is keyword and Object() is an object constructor. Object() create a reference of the object and is assigned to object name
    var fees = new Object(); equivalent to var fees = {};
  • Object constructor is a constructor function of built in object called Object.
  • Object constructor is convenient to create a single object.
