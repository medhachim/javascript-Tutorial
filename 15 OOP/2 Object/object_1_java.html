OOP: Object

  • An object is a collection of properties, and a property is an association (having a link) between name(a key) and a value(a string, an integer, a function etc). A property's value can be a function, in that case the property is known as a method. In addition to objects that predefined int he browser, you can define your own object.
    Eg:
    var fees = {
    //fees is the name of the object and below are its members
    Rahul: 100,//properties of the object fees
    Sunil: 200,//properties of the object fees
    Irene: 400,//properties of the object fees
    total: fucntion(){return(100+200+400);}// a function is used in this property and it is called the method of the object fees
    }

  • Types of objects:
    1. User defined objects: These are custom objects created by programmers to bring structure and consistency to a particular programming task.
    2. Native objects: These are provided by the JavaScript language itself like String, Number, Boolean, Function, Date, Object, Array, Math, RegExp, Error as well as object that allow the creation of user defined objects and composite types.
    3. Host objects: These objects are not specified as part of the JavaScript language but that are supported by most host environments, typically browsers like windows navigator.
    4. Document Objects: These are part of Document Object Model(DOB), as defined by the W3C. These objects presents present the programmers with structured interface to HTML and XML documents. Access to the document object provided by the browsers via the document property of the window object (window.document).
