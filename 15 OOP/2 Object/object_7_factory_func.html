Object: Factory Function

  1. When a function returns an object, we call it a factory function. It can produce object instance without using new or classes.
    Exampale:
    function mobile() {
    return {model:'Galaxy',
    price: function(){return("Price: Rs 3000");}
    };
    }
    //Creating new object instance without using 'new' keyword.
    //In other words, we have created a class 'mobile' and below 'var samsung = mobile()' is an object instance;
    var samsung = mobile();
    //mobile() is a constructor where the initialization is passed.
    //accessing:
    document.write(samsung.model+" "+samsung.price());

  2. Simple object:
    var samsung = {model:"Galaxy", price: function(){return (Price: Rs 3000");}};
    //Accessing:
    document.write(samsung.model+" "+samsung.price());
    //In this situation, we have to again create properties and methods in case of another object LG
    Therefore, factory function is the alternative to avoid the construction of properties for each model.
    var samsung = mobile(); var LG = mobile();
