Variable Block Scope

  • 1. Block is not a function block. Varible inside the if block has the global scope when declared with var keyword.
1. eg:
if(true)
{
var i = 20;
document.write(i);
}
document.write(i) //Here we can access the variable from outside the scope.
  • 2. Block is not a function block. Varible inside the if block has the local scope when declared with let keyword.
2. eg:
if(true)
{
let i = 20; //Now the variable scope is local
document.write(i);
}
document.write(i) //Here we cannot access the variable from outside the scope.
