Variable Scope

  • There are two types of variable scopes. 1. Globle Scope 2. Local Scope
  • 1. A variable declared outside the function is a global variable and its value is accessible and modifiable throught the program. In a web browser, global variables are deleted when you close the browser window(or tab), but remain available to new pages loaded into the same window.
  • 2. A variable declared inside the function is a local variable. It is created and destroyed every time the function is executed, and it cannot be by any code outside the function. Inside a function, if a variable has not been declared with var it is created as a global variable.
  • 3. Function arguments(parameters)work as local variable inside the function.
