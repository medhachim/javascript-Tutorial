Function Expression

  • When we create a function assign it to a variable, known as function expression.The block of function expression is closed with a semi-colon(;)
    eg:
    var my_func = function show()
    {
    document.write("Rose Tamang");
    };
    my_func();
  • Note:We cannot call function expression before defining a function.
    my_func();//calling fucntion expression before hand is a wrong idea. Because function expressions in Javascript are not hoisted, unlike function declaration.
    var my_func()
    {
    document.write("Rose Tamang");
    };
