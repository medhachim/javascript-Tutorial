Function in javascript.


Functions are subprograms which are used to compute a value or perform a task.
Types of Function:
1. Library or Built in function.
eg: valueOf(), write(), alert()
2. User defined function.
a) A function name starts with an underscore(_) or a letter.
b) A function name doesn't start with a number.
c) Do not use reserved words. eg: else, if etc.
d) Function names are case sensitive in javascript.
