Closure Function

  1. Closure is a function having access to the parent scope. It preserves the data from outside.
    var i = 10; // parent scope
    function show()
    {
    var j = 20;
    document.write(j);
    document.write(i); //Can access the parent scope(outside)
    // Here closure is created between the two scope;
    }
    show();
  2. Closure is an inner function that has access to outer (enclosing) function's variables.
  3. Every closure has three scopes.
    a. Local Scope (Own Scope)
    b. Outer Function's Scope
    i) The function inside the function is an inner function and the function which is outside is the outer function. The inner function can access the variable of outer function. The closure function is also called nested function.
    c. Global Scope
