Default Parameter


Syntax: function function_name(para1, para2, para3 = 'value')
{
block of statments;
}
Syntax: function function_name(para1, para2 = 'value1', para3 = 'value2')
{
block of statments;
}
Here, incase a value is not passed to para3, the default 'value' will be used. However, the passed value will have the high priority.
The default value must be assigned at the end of other parameter and not in the middle of them. Otherwise it will display 'problem undefined'.
