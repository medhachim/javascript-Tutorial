Arrow Function

An arrow function expression(previously, and now known as fat arrow function) has a shorter syntax compared to function expression. Arrow functions are always anonymous(no name). It cannot be called before defining.
Occurs:Type Error:myfun is not a function. We are not supposed to break line with fat arrow(=>)
Syntax of arrow function:
()=>{statements};
**********************
eg: of a function expression
var myfun = function show()
{
document.write("Rose Tamang");
};
1. The same function expression can be written in arrow function syntax:
2. var myfun = ()=>{document.write("Rose Tamang")};
3. This arrow function has no function name and a keyword 'function'.
4. Arrow function can be nested with other functions.
5. Arrow function can be return with other functions.
