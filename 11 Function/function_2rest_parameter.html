Rest parameter.

Difference between parameter and argument:
A variable of a function in a method is a parameter. An argument is the value assigned to the variable(parameter) in a function.
The rest parameter allows to represent an indefinite number of arguments as an array.
Rest arguments start with three dots.(...). The rest operator must be the last parameter to a function.
Syntax:
function function_name(...args)
{
Block of statements;
}
or
Syntax:
function function_name(a,...args)
{
Block of statements;
}

Difference between rest parameters and arguments object.

There are three main differences between rest parameters and arguments object.
  • Rest parameters are only the ones that haven't been given separate name,
  • The arguments object is not a real array, while Rest Parameters arre array instances, meaning methods like sort, map, forEach or pop can be applied on it directly. while the arguments object contains all arguments passed to the function.
  • The arguments object has additional functionality specific to itself(like the callee property.

  • A value is assigned to 'a' separately and the rest of them is assigned to rest parameters.
  • A value is assigned to 'a' as well as the arguments object takes all the value including that of 'a'.
