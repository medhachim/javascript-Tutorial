Null Value

1. The null value indicates an empty value; it is essentially a placeholder that represents 'nothing'. The null value is defined as an empty objects so using typeof operator on a variable holding null shows its type to be object.

    Difference between null and undefined.
  • undefined means the value hasn't been set.
    eg: var a;
  • null means the value has been set to be empty.
    eg: var a = null
